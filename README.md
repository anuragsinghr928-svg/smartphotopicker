# 🧠 Face Match → Export

A powerful browser-based tool to **match faces from bulk images** using reference photos and export results as **PDF or ZIP**.

---

## 🚀 Features

- 📸 Upload 1–5 reference photos (front + side)
- 📂 Scan bulk images from:
  - Local folder
  - Individual files
  - Google Drive (Picker + Link)
- 🧠 Face detection using AI (face-api.js)
- 🎯 Adjustable match sensitivity & profile boost
- 📊 Real-time progress & stats
- 📄 Export results as:
  - PDF (one image per page)
  - ZIP (original images)

---

## 🛠️ Technologies Used

- HTML, CSS, JavaScript
- face-api.js (AI face recognition)
- jsPDF (PDF generation)
- JSZip (ZIP creation)
- Google Drive API (optional integration)

---

## ⚙️ How It Works

1. Upload reference images (front + side recommended)
2. Select bulk image source (folder/files/drive)
3. Click **Start Face Matching**
4. Download matched results as PDF or ZIP
